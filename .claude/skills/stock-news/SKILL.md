# stock-news — 주식판 뉴스 보충 생성

수강생 가상 주식판(HK_Stock)의 재미를 위해 **테마주 뉴스 헤드라인**을 추가로 만든다.
자동 뉴스 엔진(`functions/news.js`)의 내장 템플릿을 보충하거나, 특정 이벤트용 뉴스를 생성한다.

## 1) 맥락 먼저 파악
1. 현재 상장 종목·업종·특성을 확인한다.
   - 서비스 계정 키가 있으면 `test-harness` 패턴(firebase-admin)으로 `stocks`(name, sector) + `stockTraits`(traits)를 읽는다.
   - 없으면 사용자에게 **종목명 / 업종 / 특성**을 묻는다.
2. 기존 톤을 `functions/news.js` 의 `TPL` 에서 확인한다 — 가볍고 유머러스한 한국어 신문 헤드라인체, **수업 맥락**(강사 평가·근태·과제·결과물·팀워크)과 가상 기업 운영 상황을 섞음.

## 2) 생성 규칙
- **3 스코프 × 3 극성**으로 만든다.
  - `individual` — `{기업}` 플레이스홀더, 특정 종목 대상.
  - `sector` — `{업종}` 플레이스홀더, 공개 업종 테마(동반 등락).
  - `trait` — **특성명·종목명을 절대 노출하지 않음**(비공개). "일부 종목 동반 …" 같은 일반 표현만 사용해 학생이 숨은 테마를 추론하게 한다.
  - `good`(호재) / `bad`(악재) / `flat`(중립·분위기).
- 한 문장, 12~22자, 헤드라인체. 실존 인물 비방·민감 내용 금지, 가볍고 긍정적인 게임 톤.

## 3) 출력 형식 (사용자가 원하는 쪽)
- **A. 템플릿 뱅크 추가용** — `functions/news.js` 의 `TPL` 에 그대로 붙일 수 있는 JS 문자열 배열로. 스코프/극성별로 구분해 출력.
- **B. 즉시 게시용** — 운영자 화면 **④ 수동 뉴스**로 올릴 헤드라인 목록. 시세 동반 효과가 필요하면 **③ 시세조정**과 함께 쓰라고 안내.

## 4) 주의
- 뉴스 **텍스트만** 생성한다. 시세는 직접 바꾸지 않는다(자동 엔진/`adjustPrice` 담당). 총량 보존 규칙([[hk-stock-project]])을 깨지 않는다.
- 특성(trait) 뉴스는 비공개 원칙 유지 — 특성명/대상 종목을 드러내지 말 것.
- 템플릿 뱅크에 추가할 때는 `src`/`functions` 의 기존 형식·따옴표 컨벤션을 그대로 따른다.
